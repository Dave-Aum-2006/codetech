# CODETECH Internship Tasks

This repository contains all four internship tasks completed as part of the CODETECH Internship Program using modern web development technologies.

## 📁 Tasks Included

### 🚀 Task 1 - Public API Web Application
A responsive web application that fetches and displays real-time data from public APIs such as Weather and News APIs.

### 💬 Task 2 - Real-Time Chat Application
A full-stack real-time chat application built using Socket.IO with live messaging and authentication features.

### 🌐 Task 3 - Full-Stack Dynamic Web Application
A modern full-stack application using React.js, Node.js, Express.js, and MongoDB with CRUD operations and authentication.

### 🧩 Task 4 - Chrome Productivity Extension
A Chrome extension that tracks website usage time, analyzes productivity, and provides weekly productivity reports.

## 🛠️ Technologies Used
- React.js
- Node.js
- Express.js
- MongoDB
- Socket.IO
- Tailwind CSS
- JavaScript
- Chrome Extension APIs
